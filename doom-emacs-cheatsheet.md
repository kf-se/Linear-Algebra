# Doom Emacs Cheatsheet

## Windows & Panes

| Action              | Keys      |
| ------------------- | --------- |
| Close current pane  | `SPC w c` |
| Maximize pane       | `SPC w m` |
| Shrink pane height  | `SPC w -` |
| Grow pane height    | `SPC w +` |
| Split horizontally  | `SPC w s` |

## Buffers

| Action               | Keys      |
| -------------------- | --------- |
| Kill current buffer  | `SPC b k` |
| Kill specific buffer | `SPC b K` |
| Switch to buffer     | `SPC b B` |

## Files & Navigation

| Action             | Keys      |
| ------------------ | --------- |
| Open dired         | `SPC o -` |
| Delete in dired    | `d` then `x` |
| Go to definition   | `gd`      |
| Go back            | `C-o`     |

## Terminal

| Action         | Keys      |
| -------------- | --------- |
| Open terminal  | `SPC o t` |

## Code

| Action            | Keys      |
| ----------------- | --------- |
| Compile           | `SPC c C` |
| Format buffer     | `SPC c f` |
| List diagnostics  | `SPC c x` |

## Help

| Action              | Keys        |
| ------------------- | ----------- |
| Describe key        | `SPC h k`   |
| Search keybindings  | `SPC h b b` |
| Search functions    | `SPC h f`   |

## General

| Action          | Keys      |
| --------------- | --------- |
| Restart Emacs   | `SPC q r` |
